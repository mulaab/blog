Woow… keren judulnya. Padahal penulis belum pernah tembus di jurnal tersebut… Seorang guru akan dicontoh oleh muridnya, jika guru telah melakukan apa yang dibicarakan atau disampaikian. Bagaimana menjadi seorang penulis profesional ikuti langkah langkah berikut..

  1. IKuti panduan yang telah ditetapkan oleh IEEE
  2. Hasilkan karya yang unik
  3. Publikasi atau kirim tulisan anda

Dari beberapa browsing di internet dapatlah saya langkah 1… yaitu panduan dari menulis latex menggunakan IEEE latex class judulnya “How to Use the IEEEtran LATEX Class “

Artike ini menjelaskan bagaimana menggunakan IEEEtran class dengan LATEX untuk menghasilkan kualitas tulisan makalah yang sesuai untuk mendafatar ke Institute of Electrical and Electronics Engineers (IEEE). IEEEtrans class menghasilkan makalah konferensi, jurnal, dan technical note (correspondence) papers dengan memilih opsi kelas yang sesuai.

  1. INTRODUCTION

Dengan file IEEEtran class komputer menjalankan LATEX, dan memahami dasar dari bahasa Latex, penulis dapat menghasilkan makalah penelitian yang profesional dengan cara cepat, mudah, dan sedikit upaya. Tujuan makalah ini adalah memberikan panduan dari IEEEtran LATEX class dan memberikan dokumentasi tentang fitur fitur unik dari kelas tersebut dan karakter-karakternya. Dokumen ini menggunakan versi 1.8b dan IEEE trans terbaru. Versi sebelumnya tidak memiliki semua fitur yang dijelaskan disini. IEEE akan menampilkan nomor versi pada console user ketika dokumen dikompilasi. Versi terakhir dari IEEE dan file file pendukung dapat anda temukan pada website [1], atau CTAN [2]. Website yang terakhir mungkin mempunyai beberapa materi tambahan seperti versi test beta dan file file yang terkait dengan non_EEE yang menggunakan IEEEtran. Lihat laman [3] untuk pertanyaan pertanyaan yang sering ditanyakan dan berita berita terkini tentang IEEEtrans. Contoh contoh lain dari dokumen ini adaah bare_conf.tex, bare_jrnl.tex, bare_jrnlcomsoc.tex, bare conf_compsoc.tex, bare_jrnlcompsoc.tex and bare jrnl_transmag.tex, which are “bare bones” contoh template untuk conference, journal, IEEE CommunicationsSociety journal, IEEE Computer Society conference, IEEE Computer Society journal and IEEE TRANSACTIONS ON MAGNETICS paper. Penulis dapat menggunakan file file ini untuk segera memulai sesuai kebutuhan. Contoh contoh lanjutan yang lebih komplek dapat ditemukan pada bare_adv.tex.
